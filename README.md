# US Road Network Graph Analytics

## Description
This project analyzes the US Road Network using Graph Theory and Neo4j. It includes data processing, shortest path algorithms (Dijkstra), centrality analysis, and a Plotly dashboard for network metrics.

## Files
- **process_data.py**: Converts raw `usa.txt` into CSV format for Neo4j import.
- **dashboard.py**: Generates a KPI dashboard using Plotly.
- **Assignment 2 Report.pdf**: Full documentation of the methodology and results.
